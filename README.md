# Eepy URL

A simple URL shortener made with React and FastAPI.

## Tech Stack

### Frontend

1. React
2. Tailwind CSS
3. TypeScript
4. React Router
5. SWR
6. shadcn/ui

### Backend

1. FastAPI
2. SQLite
3. SQLModel
4. PyJWT

## Run

```bash
docker compose build --no-cache
```

```bash
docker compose up -d
```
